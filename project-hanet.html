HANet: Hierarchical Augmentation for Continual Few-shot Event Detection

Abstract

Continual few-shot event detection (CFSED) aims to learn new event types from a few examples while maintaining performance on previously learned types. This is challenging due to the severe "catastrophic forgetting" problem in few-shot scenarios.

HANet proposes a Hierarchical Augmentation framework that leverages both data-level and feature-level augmentation to enhance the model's robustness and memory. By applying hierarchical strategies, we can effectively bridge the gap between base classes and novel few-shot tasks.

Key Contributions

  • Introduced Hierarchical Augmentation to improve feature diversity in low-resource settings.
  • Mitigated catastrophic forgetting using a balanced memory replay strategy.
  • Achieved state-of-the-art results on the MAVEN dataset for continual event detection.
