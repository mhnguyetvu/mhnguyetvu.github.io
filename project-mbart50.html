English-Vietnamese Translation with mBART50

Overview

This project focuses on fine-tuning the mBART50 model for the English-to-Vietnamese translation task. mBART50 is a many-to-many multilingual sequence-to-sequence model pre-trained on a large-scale corpus of 50 languages.

Our goal is to optimize the model for better fluency and accuracy in Vietnamese translations, especially in technical and formal domains.

Objectives

  • Adapt the pre-trained mBART50 model for the En-Vi language pair.
  • Utilize techniques like teacher forcing and label smoothing to improve convergence.
  • Evaluate performance using BLEU and TER scores across multiple test sets.

Technologies

Hugging Face Transformers PyTorch mBART50-large-50 Tokenization
