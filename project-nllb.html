NLLB-200 Fine-tuning for Korean-Vietnamese Translation

Project Overview

This project implements a comprehensive pipeline for fine-tuning Facebook's NLLB-200-distilled-600M model on Korean-Vietnamese parallel data, achieving +20% BLEU improvement over the baseline through advanced quality filtering and semantic alignment techniques.

The key innovation lies in the multi-stage filtering pipeline combined with automated threshold optimization, demonstrating that quality significantly outweighs quantity in neural machine translation training data.

Multi-Stage Filtering

  • Basic Filtering: Length, ratio, special characters removal
  • Enhanced Filtering: Language ID verification (80% confidence), near-duplicate detection (MinHash LSH), boilerplate removal
  • Semantic Alignment: LaBSE embeddings for cross-lingual similarity scoring

Threshold Optimization

  • Automated sweep across multiple similarity thresholds (0.65, 0.70, 0.75, 0.80)
  • Quick training (5K steps) for rapid comparison
  • Comprehensive evaluation: BLEU, chrF, TER, COMET metrics

Training Pipeline

  • Automated train/dev/test splitting (95/2.5/2.5)
  • Production training with gradient checkpointing and fused optimizer
  • Best checkpoint selection based on validation metrics

Key Finding

Quality > Quantity:

27K high-quality pairs (threshold 0.80) outperformed 59K medium-quality pairs (threshold 0.65) by +20% BLEU, validating the importance of semantic alignment in translation data.
